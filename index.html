Visual Clothing Recommender

Python · PyTorch · HuggingFace Transformers · ResNet-50 · Streamlit

A computer vision app that recommends visually similar clothing items from a fashion catalogue. Upload a photo of an item and get back the closest visual matches, ranked by similarity.

  • Uses a pre-trained ResNet-50 model to generate a 2048-dimensional embedding for every catalogue image and the uploaded photo.
  • Ranks matches using cosine similarity between embeddings.
  • Dataset cleaned to remove invalid and duplicate images via file hashing before embedding.
  • Deployed on Streamlit Community Cloud with image assets hosted on Hugging Face.

Lando — Compliance-as-a-Service

Python · Flask · Google Gemini Pro · Stripe API

A full-stack audit platform that checks UK tenancy agreements for compliance with the Renters’ Rights Act (2025/2026), returning results in seconds.

  • Privacy-first architecture using UUID-based file isolation with no long-term data retention, aligned with GDPR.
  • End-to-end Stripe Checkout monetisation flow, including automated invoice generation.
  • Prompt engineering and compliance logic iterated using a structured pass/fail test framework built from legal research.
